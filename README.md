# gesture-based-multi-mode-interface

A real-time computer vision application that enables touchless computer interaction using hand gestures captured through a webcam. The system uses OpenCV and MediaPipe for hand tracking and gesture recognition and provides multiple interaction modes for controlling different computer functions.

## Features

* Real-time Hand Tracking
* Gesture Recognition
* Cursor Control Mode
* Drawing Mode
* Media Control Mode
* Presentation Navigation Mode
* Streamlit-Based Interface
* Stability Filtering for Improved Accuracy

## Technologies Used

* Python
* OpenCV
* MediaPipe
* Streamlit
* NumPy
* PyAutoGUI

## Interaction Modes

### Cursor Mode

* Move cursor using index finger
* Perform mouse clicks using gestures

### Drawing Mode

* Draw on a virtual canvas
* Select colors and tools through gestures

### Media Mode

* Control volume
* Play/Pause media
* Navigate tracks

### Presentation Mode

* Navigate presentation slides
* Pause or exit presentations using gestures
